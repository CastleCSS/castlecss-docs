How it works

Every tab-header is connected to a tab-panel. To open the correct tab-panel when the user clicks on a tab-header, we use two attributes: Each tab has an attribute data-castlecss-tab-header with a value. The value of this attibute should match the value of the attribute data-castlecss-tab-panel on the tab-panel.

Options

Full width tabs

tabs are default shown inline. With CastleCSS Tabs it is also possible to make the tabs 100% width. To do so change the tabs-header to tabs-header tabs-header-full
