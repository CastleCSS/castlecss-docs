Variables

Tweak your castles defaults

Note: Never overwrite the castlecss-core variables, but instead create your own document and overwrite the variables there. We have already included said document in the full package.

Below you can find the default settings of a CastleCSS website.

							/* 	Variables 
								Add or replace variables in your OWN variables file
							*/
